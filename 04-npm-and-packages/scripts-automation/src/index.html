Build Process

This application was built using the following npm scripts:

  • npm run build - Builds all assets
  • npm run build:js - Processes JavaScript files
  • npm run build:css - Processes CSS files
  • npm start - Starts the server
  • npm run dev - Starts the development server with auto-reload
